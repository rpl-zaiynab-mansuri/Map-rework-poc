# POC - Using Next.js & Tailwind CSS


### 1\. Clone this Repository

```bash
git clone git@github.com:rpl-zaiynab-mansuri/Map-rework-poc.git
```

### 2\. Navigate to the directory

```
cd Map-rework-poc
```

### 3\. Install dependencies

```
yarn install
# or
npm install
```

### 5\. Run Dev Server

```
yarn dev
# or
npm run dev
```
